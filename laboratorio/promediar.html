Análisis estadístico de mediciones

Aqui se ha instalado un applet que encuentra las siguientes funciones:

  1. El promedio
  2. La desviación estandar
  3. La desviación estandar en la media
  4. El valor máximo
  5. El valor mínimo
  6. La media
  7. La moda

El applet tiene dos ventanas:

  • Datos. En esta ventana el usuario escribe los datos de entrada. Un dato por línea. También se puede escribir copiando los datos de algún archivo y pegandolos.

  • Resultados. En esta ventana el applet escribe sus resultados.

El applet ademas tiene dos botones:

  • Calcular. Después de capturar los datos, se debe utilizar este botón para que el applet haga sus cálculos.

  • Limpiar. Con este botón, el applet limpia las ventanas de datos y de resultados y espera nuevos datos.
